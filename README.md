#  Project : World Cup Database

##  About the Project

**Objective** : Build a PostgreSQL database to store and query data about World Cup matches and teams. This project is part of FreeCodeCamp’s Relational Database Certification and is completed using a Linux environment with PostgreSQL and Bash.

##  Project Tasks

- Create a PostgreSQL database named `worldcup`
- Define tables for `teams` and `games`
- Use appropriate data types and constraints (`SERIAL`, `INT`, `VARCHAR`, `FOREIGN KEY`, etc.)
- Insert match data for multiple World Cup tournaments
- Write SQL queries to:
  - Count total goals
  - Calculate average goals
  - Find the winning team of each tournament
  - List teams that played in specific rounds
  - Display match details with joins
- Use shell scripts to automate queries and test results
- Pass all automated tests provided by CodeRoad
